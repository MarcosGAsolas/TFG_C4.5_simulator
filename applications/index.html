Decision Tree Simulator (C4.5 algorithm)

Decision Tree Simulator using algorithm C4.5

This simulator is designed to learn and understand the development of the decision tree built by the C4.5 algorithm.
It not only teaches the logic behind the tree, but also provides tools to understand the most relevant theoretical aspects of the entire algorithm, with the goal of fully understanding how the C4.5 algorithm works.
